Clean up environment
====================

The promise of public cloud is not only to bring up application environments quickly, consistently and with minimum capex, but also to tear down application environments cleanly and completely.

- From Big-IQ delete app1
- From Big-IQ delete app2
- From Big-IQ delete userxx-ssg1
- In the AWS console, delete VPN connection
- In the AWS console, delete Customer Gateway
- From Super-NetOps terminal run terraform destroy
